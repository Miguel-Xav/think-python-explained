# think-python-explained
A repository with .py files to show the solutions of Think Python 3th Edition exercises with the explanation of each line!

## How it works and who it's for
On this repository you can see each exercise solved by me, with different methods, on the time i've read this book.

I focused on explaining each line of code to help begginers, who are facing difficulties on the exercices, understand the programming logic of Python.

The use of .py files, and not .ipynb like the most repositories with the solution, is to have an easy visualization in the github, without the necessity to download anything.

**I'm not a professional and I did this project purely for fun, to document my progress with the book and help people who are struggling. I'm open to discussing any criticism and exploring different ways to solve the exercises.**

It will have a folder for each chapter and a .py file for each exercise and a png file with a diagram explaining the logic created in [Excalidraw](https://github.com/excalidraw/excalidraw).
